This directory contains files to be used for taking encoder data using the Arduino and processing that data

(0) location.py: stores the location of the encoder data, relative to this directory
    $masterDir

(1) encoderDAQ.py: takes data from the Arduino for a given number of seconds and stores the IRIG and Encoder data into a directory named $masterDir/[Run Name]/rawData/
    Usage: python encoderDAQ.py [Run Name] [Number of seconds to collect data]

(2) encoderDataProcess.py: takes a run name for IRIG and Encoder data taken using 'encoderDAQ.py' and generates an angle vs time file at $masterDir/[Run Name]/Data/
    Usage: python encoderDataProcess.py [Run Name]

(3) encoderDataAnalyze.py: takes a run number to find the angle vs time file generated by 'encoderDataProcess.py' and generates several FFTs, TODs, etc at $masterDir/[Run Name]/Data/
    Usage: python encoderDataAnalyze.py [Run Name]
